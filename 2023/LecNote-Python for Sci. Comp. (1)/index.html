    [Quant] 证券投资学笔记

    发表于 2023-02-22 更新于 2023-11-07

    除了风险，金融市场没有什么是确定的。

    投资学关注的问题

    “定价问题”——对给定的风险资产进行定价；

    “投资问题”——在投资的风险和收益之间进行权衡对风险资产进行合理的配置；

    “消费和投资问题”——对人生不同阶段的消费与投资进行合理规划；

    “资产负债管理问题”——具有负债约束下的资产配置与投资问题。

    1.1 实物资产与金融资产

    一个社会的物质财富最终取决于社会经济的生产能力，这种生产能力是经济体中实物资产的函数——金融资产归根结底是底层的实物资产支撑的。

    实物资产为经济创造净利润，而金融资产确定收益或财富在投资者之间的分配，代表持有者对实物资产所产生收入的索取权。

    投资者购买企业发行的证券，企业就可以用筹集到的资金购买实物资产，如厂房、设备、技术和原料，而投资者虽然没有自己的汽车厂，仍然可以通过购买福特汽车的股份来分享汽车生产所产生的收入。

    投资者投资证券（金融资产）的收益最终来源于企业通过发行证券所筹集的资金购买实物资产所产生的利润。

    1.2 金融资产

    金融资产通常可以分为三类：固定收益型金融资产、权益性金融资产、衍生金融资产。

    固定收益型金融资产/债务型金融资产：承诺支付一系列固定的现金流。

    公司债券向证券持有者承诺每年固定的利息收入，除非债券发行方违约，否则债券持有者将获得固定收益——固定收益型金融资产的收益受发行方财务状况的影响最小。

    普通股/权益型金融资产：代表证券持有者对公司的所有权。

    权益型证券持有者未被承诺任何的特定收益，但是可以获得公司分配的股利，并按相应的比例拥有对公司实物资产的所有权——权益投资绩效与公司运营成败密切相关。

    衍生证券（期权/期货合约/etc.）：收益取决于标的资产的价格变化。

    衍生证券最主要的用途之一就是规避风险，即通过衍生品的投资可以将风险从一方转移给其他方，被普遍看作和用作对冲风险的管理工具。

    1.3 金融市场和经济

    金融市场的信息作用

    在市场经济中，证券价格通常在资本配置中发挥着重要的作用，引领着资本流向最具有增长潜力的行业和企业。

    股价是投资者对公司当前业绩和未来前景的综合评价。但是市场不总是理性的，从而通过股票市场来配置资本的过程有时是有效的、有时是无效的。没有理由认为市场百分百是正确的。

    风险分配

    金融市场和在金融市场上交易的各种金融工具可以使偏好风险的投资者承担风险，是厌恶风险的投资者规避风险。

    针对不同风险偏好的投资者进行筹资的方式提高了市场发挥其配置资本功能的效率。类似于在日常生活中常见到的把整件货品依其品质分割成不同的商品卖给顾客能卖得更快更好一样。

    所有权与经营权的分离

    金融市场种种丑态与争议的万恶之源。

    2012年通用电气公司的房地产、厂房、设备等资产的总价值为700亿美元，资产总额约为6850亿美元，公司拥有50多万个股东，如此众多的股东们显然不可能每一个人都参与到公司的日常管理中。

    实际做法是共同挑选一个董事会，由董事会负责聘请并监督公司的管理层。意味着公司的所有者与管理者由不同的人构成，即所有层与管理层分离。
